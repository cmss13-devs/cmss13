/obj/effect/client_image_holder

Simple effect that holds an image to be shown to one or multiple clients only.

Pass a list of mobs in initialize() that corresponds to all mobs that can see it. Code ripped from tgstation, though this has missing parts. Original path is 'code/modules/hallucination/_hallucination.dm'
