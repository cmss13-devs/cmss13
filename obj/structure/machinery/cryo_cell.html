ui_interact
(/mob/user, ui_key, /datum/nanoui/ui, force_open)

The ui_interact proc is used to open and update Nano UIs If ui_interact is not used then the UI will not update correctly ui_interact is currently defined for /atom/movable (which is inherited by /obj and /mob)

@param user /mob The mob who is interacting with this ui @param ui_key string A string key to use for this ui. Allows for multiple unique uis on one obj/mob (defaut value "main") @param ui /datum/nanoui This parameter is passed by the nanoui process() proc when updating an open ui

@return nothing
